## Objective
Responsive real estate search page using Vue JS based on the provided Figma design.

## Features
- Responsive layout (Desktop & iPhone 14 Pro Max)
- Property listing cards with image slider
- Hover animations & transitions
- Lazy loading images
- Map section
- Clean UI/UX

## Tech Stack
- Vue JS
- HTML5
- CSS3

## How to Run Project
1. Download or clone the repository
2. Open terminal in project folder
3. Run:
   npm install
   npm run dev

(If static)
Simply open index.html in browser.
